# Contributing to Basic Scientific Laws for AI Agents TD1

Thank you for your interest in contributing to this project.

## Guidelines

1. Ensure reproducibility of all scientific claims
2. Provide rigorous mathematical definitions
3. Include practical examples and code
4. Add comprehensive test coverage
5. Cite all references appropriately

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request with detailed description

## Quality Standards

- All code must be reproducible
- Mathematical definitions must be rigorous
- Examples must demonstrate practical application
- Tests must validate all claims

## Questions?

Open an issue or contact the maintainers.
